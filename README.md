## PySpark: Tratamento de Dados e Big Data

O objetivo desse projeto é demonstrar conhecimentos em PySpark.
Para esse projeto, utilizei os seguintes datasets de domínio público de estatísticas do Youtube: https://www.kaggle.com/datasets/advaypatil/youtube-statistics/ & https://www.kaggle.com/datasets/datasnaek/youtube-new?select=USvideos.csv

#### Instruções

Todos os arquivos estão com seus outputs originais, basta abri-los. Caso queira testar em tempo real é necessário baixar os datasets originais através dos links acima, clonar ou baixar esse repositório e rodar na sua máquina local (o Spark precisa estar instalado e configurado corretamente) ou através do Google Colab. 

## PySpark: Data Processing and Big Data

The objective of this project is to demonstrate knowledges in PySpark.
For this project, I utilized the following public domain Youtube statistics Dataset: https://www.kaggle.com/datasets/advaypatil/youtube-statistics/ & https://www.kaggle.com/datasets/datasnaek/youtube-new?select=USvideos.csv

#### Instructions

All files have their original outputs, just open them. In case you want to test them in real time it’s necessary to download the original datasets from the links above, clone or download this repository, and run it on your local machine (Spark must be installed and properly configured) or via Google Colab.
